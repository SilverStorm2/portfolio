# AGENTS.md — Portfolio Project Agents

Każdy agent odpowiada za jedno zadanie. Pracujemy w trybie PR → review → merge.

## Agent 0 — Repo Master

- Zakłada repozytorium Next.js + Tailwind + shadcn
- Konfiguruje TypeScript, ESLint, Prettier
- Dba o jednolity standard kodu

## Agent 1 — Layout & Navigation

- Tworzy layout strony z Navbar i Footer
- Dodaje przełącznik języka (EN/PL)
- Odpowiada za mobile-first design

## Agent 2 — Hero Builder

- Implementuje sekcję Hero z imieniem, tagline i CTA
- Dba o animacje i pierwsze wrażenie

## Agent 3 — Bio Writer

- Tworzy sekcję "O mnie"
- Dodaje zdjęcie profilowe i listę umiejętności
- Zapewnia prostą edytowalność treści

## Agent 4 — Projects Curator

- Implementuje siatkę projektów
- Ładuje dane z pliku JSON/YAML
- Upewnia się, że karty są responsywne i linkują do GitHuba/demo

## Agent 5 — Contact Manager

- Dodaje formularz kontaktowy
- Waliduje pola (imię, email, wiadomość)
- Konfiguruje wysyłkę (np. Formspree)

## Agent 6 — Accessibility & i18n Specialist

- Wdraża tłumaczenia (EN/PL)
- Przeprowadza audyt dostępności
- Dodaje aria-labels i focus states

## Agent 7 — Deployment & SEO

- Wdraża projekt na Vercel
- Dodaje meta tagi SEO i Open Graph
- Sprawdza wyniki Lighthouse
